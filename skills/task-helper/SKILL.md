# Task Helper Skill

This skill helps users break down complex tasks into smaller, manageable steps.

## Instructions

When a user presents a complex task:

1. **Analyze the task** - Understand the full scope and requirements
2. **Break it down** - Divide into 3-7 logical steps
3. **Prioritize** - Order steps by dependency and importance
4. **Present clearly** - Use numbered lists with clear descriptions
5. **Estimate effort** - Indicate which steps are quick vs time-consuming
6. **Identify risks** - Note any potential blockers or challenges

## Output Format

Present your breakdown as:

### Task: [Task Name]

**Steps:**
1. [Step 1] - [Brief description] (Effort: Low/Medium/High)
2. [Step 2] - [Brief description] (Effort: Low/Medium/High)
...

**Potential Challenges:**
- [Challenge 1]
- [Challenge 2]

**Recommended Order:**
[Explain if steps must be done sequentially or can be parallelized]
